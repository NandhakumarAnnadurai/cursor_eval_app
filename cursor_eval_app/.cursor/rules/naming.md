# Naming Rules

## Variable Naming
- All variables must use lowerCamelCase.
  Examples: userName, itemList, selectedIndex
- Private variables must start with underscore (_)
  Example: _controller

## File Naming
- Files must use snake_case.
- page file  → page.dart
- bloc file  → bloc.dart
- event file → event.dart
- state file → state.dart
- reusable widget file → widget_name_widget.dart
- api service file → api_service.dart
- repository file → *_repository.dart

## Class Naming
- Use UpperCamelCase for all classes.
  Example: LoginBloc, NotesState, ApiService, NotesRepository

These rules MUST be followed exactly.
